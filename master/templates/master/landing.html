Привет! Меня зовут Глеб!

Вы находитесь на лендинге инженерного проекта 2021. И называется проект Clean Flat. В процессе разработки был изучен интереснешний язык программирования Python совместно с фреймворком Django. Была изучена архитектура фреймворка, схема работы с базами данных, аутентификация и многое другое!

Формулировка задач

Задачей инженерного проектирования является создание сервиса для заказа уборки квартиры с помощью фреймворка Django. Для этого нужно проанализировать существующие решения отечественных и зарубежных сайтов, проанализировать их структуру и функциональность. Следующим этапом будет разработка инфологической и физической схемы базы данных, разработать кейсы использования приложения разными пользователями, администратором или оператором.

Сущности базы данных
  • Пользователь
    • Клиент
    • Администратор
  • Услуга
  • Заявка
  • Отзыв
  • Жалоба
Сценарии использования
У клиента должна быть возможность просматривать информацию на сайте, создавать заявки на уборку, оставлять отзывы и жалобы. Единственной необходимой функцией у клинера является только просмотр списка полученных заявок на уборку. Всем остальным занимается администратор. Администратор модерирует всю информацию на сайте. Просматривает список клинеров, просмотров, жалоб, услуг, вопрос-ответов, скидок; редактирует удаляет объекты списка. Также администратор назначает клинеров на нераспределенные заявки.
